# 微信小程序（已归档）

本项目主路线为 **Web 原型（React + Vite）+ Express 后端 + 后续真实硬件（`src/hardware/`）**。

微信小程序代码已于 2026-06 移入此目录，不再维护。如需恢复，用微信开发者工具打开本目录即可。

## 目录说明

| 路径 | 说明 |
|------|------|
| `app.js` / `app.json` | 小程序入口与页面配置 |
| `pages/` | 患者/医生/家属页面 |
| `utils/` | 小程序 API 封装 |
| `scripts/start-experience.sh` | 体验版公网隧道（localtunnel/ngrok） |

## 微信开发者工具缓存清理（可选）

若本机仍打开过旧项目，可在微信开发者工具中：

1. **工具 → 清缓存 → 全部清除**
2. 关闭或删除指向项目根目录的旧小程序工程
3. 若需再次打开小程序，请选择 `_archive/miniprogram` 作为项目目录
